Robot gekko limpia paneles solares 9 de mayo del 2019

¿En que consiste el robot gekko?

  • El robot GEKKO Solar Farm está diseñado para granjas solares de escala industrial.
  • Su mayor reto son las dimensiones; una solución de limpieza automatizada es indispensable.
  • La suciedad es una cuestión económica, mucho más importante que para las plantas fotovoltaicas de instalaciones residenciales.
  • Con una eficiencia de hasta 2.000 m2 el robot es muy potente.
  • El robot se adapta a diferentes anchos de filas de paneles; el número de cepillos y el soporte reducen fácilmente.
  • El GEKKO Solar Farm funciona en inclinaciones de hasta 30 °.
  • Es capaz de superar obstáculos de 60 cms de ancho.
  • El usuario lo dirige a control remoto y joystick.
  • Con el vehículo opcional de suministro el robot se vuelve independiente: la fuente de alimentación de energía, el aire y el agua a presión están integrados en el vehículo. Una grúa mueve el robot de una fila de paneles a otra.
  • Este gran robot también actúa en base del principio GEKKO.

    Ventajas del robot gekko

    Las ventajas y beneficios de este robot a tener en cuenta.

  • Productividad convincente: hasta 2.000 m2 por hora.
  • Sin instalaciones adicionales: el robot utiliza los mismos paneles.
  • Ancho de limpieza variable: cantidad variable de cepillos
  • Manejo fácil: control remoto y joystick
